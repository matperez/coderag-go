# CodeRAG-Go

Гибридный поиск по кодовой базе (ключевой + опционально векторный) в виде одного Go-бинарника с MCP-сервером для интеграции с Cursor, Claude и др.

- [План реализации](docs/IMPLEMENTATION_PLAN.md) — пошаговый план с тестами и коммитом после каждого шага.
